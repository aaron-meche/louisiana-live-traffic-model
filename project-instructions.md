# Louisiana Live Traffic Model

Real-time traffic analysis platform for Louisiana highways using 511la.org camera feeds, computer vision (YOLOv8), and interactive map overlays.

## Objective

To develop a codebase that has the ability to fetch public 511LA.org traffic cameras on highways within Louisiana, process the live camera feed data by utilizing Machine Language and other AI abilities to count the number of vehicles that pass a specific traffic camera with the ability to distinguish between both directions of traffic, updating a live traffic data model for purposes of enhanced traffic data, such as the types of vehicles driving on a particular type of road, estimated wear and tear on infrastructure, frequency of vehicles taking a particular exit, average vehicles for any given time, and so on.

## Purposes

This project is a personal project due to my own interest in road infrastructure and software engineering. This will be a project to be added to my resume, and future purposes may include sharing the software with public entities such as parish or state level agenencites for enhanced information regarding their roadways and ideal repair intervals
